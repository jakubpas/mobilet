Program generuje raport parkowań aplikacją mobilet w sobote w Pozaniu
=====================================================================

1. Logujemy sie na swoje konto mobilet
2. Kopiujemy swoja historę biletów do pliku tekstowego  jak na przykładzie (mobilet.csv)

Generujemy raport: 

`php mobilet.php`

I wysyłamy do ZDM

U mnie wyszło 15zł ;-)
